Estudando agora:

Markdown:
Markdown é uma linguagem simples para formatação de texto em plataformas digitais, com símbolos que facilitam adicionar elementos como títulos, listas e links. Popular entre escritores e desenvolvedores pela sua portabilidade e facilidade de uso.
HTML:
HTML é a linguagem fundamental para a criação de páginas web, utilizando tags para estruturar elementos como texto, imagens e links. Essencial para o desenvolvimento na web devido à sua simplicidade e ampla compatibilidade.
CSS:
CSS é uma linguagem de estilo para controlar a aparência de páginas web, separando o conteúdo do design. Permite definir layout, cores e fontes, facilitando a criação de sites estilizados e consistentes.
